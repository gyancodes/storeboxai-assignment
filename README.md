# File Explorer

A minimal, VS Code-style file management system built with React, TypeScript, and Tailwind CSS.

## Features

- **Nested Explorer**: Recursive file and folder structure.
- **CRUD Operations**: Create, rename, and delete files/folders.
- **Minimal Design**: Pure blue and white aesthetic.
- **Zero Libraries**: Built from scratch without external tree libraries.

## Setup

```bash
# Install dependencies
npm install

# Run locally
npm run dev
```

## Tech Stack

- React 19
- TypeScript
- Tailwind CSS
- Lucide React (Icons)
